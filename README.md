# Wizard Chess — Harry Potter Edition

A Harry Potter-inspired Wizard Chess game built in C++ with Hogwarts-themed graphics.

## Features
- Complete chess logic with move validation and special rules
- VS AI mode with a self-built AI function
- Hogwarts-inspired custom graphics and atmosphere
- Built with Raylib
